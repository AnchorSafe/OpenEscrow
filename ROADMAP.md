# OpenEscrow Roadmap

## v0.1 — Foundation ✅ (current)

- [x] Soroban escrow contract (create, fund, milestones, disputes, refund)
- [x] Express REST API with Stellar SDK integration
- [x] React dashboard (create, fund, track, dispute, resolve)
- [x] PostgreSQL off-chain state mirror
- [x] Public agreement pages (shareable links)

## v0.2 — Wallet & UX

- [ ] Freighter wallet integration (replace raw secret key input)
- [ ] Albedo and xBull wallet support
- [ ] Email/webhook notifications on milestone events
- [ ] Mobile-responsive improvements
- [ ] Escrow search and filtering

## v0.3 — Multi-party & Templates

- [ ] Multi-signature escrow (2-of-3 approval)
- [ ] Escrow templates (freelance, agency, SaaS, export)
- [ ] Recurring milestone schedules
- [ ] Deadline enforcement (auto-refund on expiry via time-locked contract)

## v0.4 — Decentralised Dispute Resolution

- [ ] On-chain arbitration pool (community mediators)
- [ ] Mediator reputation system
- [ ] Evidence submission (IPFS-linked)
- [ ] Dispute fee mechanism

## v0.5 — Integrations

- [ ] Zapier / Make webhook triggers
- [ ] Slack and Telegram notifications
- [ ] Invoice generation (PDF)
- [ ] Accounting export (CSV, QuickBooks)

## v1.0 — Production

- [ ] Mainnet deployment guide
- [ ] Audit by a Soroban security firm
- [ ] SDK / npm package for embedding escrow in other apps
- [ ] Multi-language UI (ES, PT, FR, AR)
- [ ] Hosted SaaS option with custom domain support
